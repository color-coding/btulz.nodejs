# btulz.nodejs
这是Node.js环境下的开发工具集。

## 安装说明 | install
* nodejs环境下：
~~~
npm install -g btulz
~~~

## 使用说明 | instructions
### types-ui5 生成OpenUI5库文件
~~~
btulz types-ui5 v1.61.2
~~~
* 参数1（data）：可以是说明文件目录或文件；也可是版本号，版本号时自动从网站下载说明文件。
* 参数2（out）：生成的库文件输出目录，默认为当前目录。
